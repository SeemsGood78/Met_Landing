# The MET — Landing Page

This project is a front-end implementation of a landing page for **The MET**, based on the Figma design. It focuses on translating a complex layout into a functional web page using modern CSS techniques.

## Live Preview
**Live Demo:** [View the live website here](https://seemsgood78.github.io/layout_landing-page/)

## Design Reference
**Figma Link:** [The MET Design](https://www.figma.com/file/lSR1m42L9YwzQwzzxKwHpw/THE-MET)
*Note: The implementation is a simplified version of the design, focusing on structural accuracy rather than pixel-perfection.*

## Technologies Used

**Core**
* **HTML5** — Semantic markup.
* **SCSS** — Utilizing mixins and variables for layout control.
* **BEM Methodology** — Consistent and scalable CSS naming.

**Development & Deployment**
* **Vite** — Fast build tool and development server.
* **GitHub Pages** — Hosting and deployment.

## Getting Started

Follow these instructions to set up the project locally:

1. **Clone the repository:**
```bash
git clone https://github.com/SeemsGood78/layout_landing-page.git
cd layout_landing-page
```

2. **Install dependencies:**
```bash
npm install
```

3. **Run the project locally:**
```bash
npm run dev
```

## Features
* **Adaptive Layout, Tablet Support and Desktop Optimization:** Optimized layout transitions using custom SCSS mixins, providing a tailored experience for Tablet (min-width: 744px) and Desktop (min-width: 1260px) resolutions.
* **Clean Codebase:** Focused on a simple and readable structure without unnecessary complexity or external libraries.
